# Nerf-Maverick-Rev-6-Reverse-Engineering-Project
Led a team of 4 mechanical engineers to reverse engineer a Nerf Maverick blaster. Fully disassembled and measured every component using a digital caliper for precise dimensions. Modeled all parts in Siemens NX and assembled them with constraints to simulate motion. Analyzed trigger linkage, cylinder indexing, and air plunger compression behavior.
